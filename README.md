# Attendance Percentage Calculator

A simple and user-friendly web application to calculate student attendance percentage and determine how many additional classes are required to meet a minimum cutoff.

This tool helps students plan their attendance effectively and avoid last-minute surprises.

---

## Features

- Calculate current attendance percentage  
- Determine minimum number of additional classes required to meet cutoff  
- Clear validation and error messages for invalid inputs  
- Handles important edge cases such as:
  - Attended classes exceeding conducted classes  
  - Conducted classes exceeding total classes  
  - Cutoff already satisfied  
- Responsive and clean user interface  
- Single-page experience with instant results  

---

## Tech Stack

- React (Functional Components and Hooks)  
- Tailwind CSS  
- JavaScript  

---

## How It Works

The user provides:
- Total number of classes scheduled for the semester  
- Number of classes conducted so far  
- Number of classes attended so far  
- Minimum attendance cutoff percentage  

The application:
- Validates all inputs  
- Calculates the current attendance percentage  
- Computes the minimum number of additional classes required to maintain the cutoff by the end of the semester  

Results are displayed instantly without page reload.
